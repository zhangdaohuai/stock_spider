# 基础实时数据爬虫

## 1. 核心功能
- **分钟级别 A股 全量数据 爬取**
- **特化标准**：ST、退市风险、同时有港股等
- **五手买卖信息**
- **数据来源等**

## 2. 数据要求
- 数据回溯到 **2016年1月1日** 开始，如果第一次爬取，需要向前回溯到该日期

## 3. 数据接口
- 使用以下两个接口：
  - [AkShare](https://akshare.akfamily.xyz/introduction.html)
  - [Baostock](https://baostock.com/mainContent?file=pythonAPI.md)

### 注意：
1. 需要支持**多账号**，每个数据源都可能有多个账号
2. 需要先研究确认各个接口的**边界**：如调用次数、频率等
3. 当程序运行时，需要先检查数据是否完整，不完整，需要通过接口进行补充完整
4. 监控实盘网页功能需要先调研最佳实现方式，考虑是否需要使用 **Selenium** 等工具进行模拟浏览器操作，必须先行验证，验证合格的代码不能删除，放入tests目录下的functional_test_case目录


## 4. 配置管理
- 账号等配置使用 **JSON 格式** 文件

## 5. 数据存储
- 数据存入 **PostgreSQL**

## 6. 开发环境
- 开发语言：**Python**
- 开发及运行环境：使用 **conda** 中 `agent` 环境

## 7. 界面要求
- **无界面设计**

## 8. 运行环境
- 需要可以在 **Mac**、**Ubuntu** 环境下**后台运行**，**无人值守长期自动运行**

## 9. 日志与告警
- 日志**严格分级**，严重错误通过**邮件**及时通知

## 10. 实时监控
  1、- 实时实盘信息监控（**每分钟刷新**）
    - [https://quote.eastmoney.com/zs000001.html](https://quote.eastmoney.com/zs000001.html) （上证指数）
    - [https://quote.eastmoney.com/zs399001.html](https://quote.eastmoney.com/zs399001.html) （深证成指）
  2、- 全面分析 这两个网页及其所有子目录的所有数据
      - 1、提取 数据并形成 **DataFrame** 已经 页面及其所有子目录的所有数据 分析文档 和 对应的下载及存储方案
      - 2、分析 数据，为长期监控并批量存储准备
      - 3、通过分析，形成下载测试用例
